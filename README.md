# Frontend – CRUD de Usuários

Frontend desenvolvido utilizando **React.js**, que permite gerenciar usuários com interface interativa para criação, listagem, atualização e exclusão.

## Funcionalidades
- Exibir lista de usuários
- Criar novos usuários via formulário
- Atualizar dados de usuários existentes
- Deletar usuários diretamente da interface

## Tecnologias Utilizadas
- [React.js](https://reactjs.org/) — biblioteca JavaScript para construção de interfaces
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) — linguagem principal da aplicação
- [Axios](https://axios-http.com/) — biblioteca para requisições HTTP
- [Vite](https://vitejs.dev/) — ferramenta de build e desenvolvimento rápido
